# 计算分子生物学

**课程简介**：学习生物信息学中的核心算法与计算方法。

---

## 📚 课程内容

本课程主要内容包括：

- 🧬 **序列比对** - 双序列比对、多序列比对算法
- 🌳 **系统进化树** - 进化树的构建方法与分析
- 📊 **概率模型** - 马尔可夫链及其应用
